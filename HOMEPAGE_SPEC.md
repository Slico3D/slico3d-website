# SLICO3D Homepage Specification

## Goal

Create the first public homepage for SLICO3D.

The homepage should not feel like a simple business card.

It should feel like a living brand hub with:

* Clear branding
* Latest news
* Product highlights
* Blog connection
* eBay shop link
* Contact options

---

## General Style

Use a clean, modern and friendly design.

Default design should be light.

Support automatic dark mode based on user device preferences.

The website should feel:

* modern
* technical
* trustworthy
* friendly
* made in Germany / Bavaria

Inspired by:

* Apple
* Notion
* Stripe
* Ubiquiti

Avoid:

* gaming look
* cyberpunk design
* overloaded layouts
* excessive gradients
* childish 3D-printing style

---

## Homepage Flow

The homepage should scroll naturally.

Structure:

1. Hero
2. Latest News
3. Product Highlights
4. Made in Bavaria
5. Blog / Workshop Updates
6. Contact

---

## 1. Hero Section

Main visual element:

Use the official SLICO3D logo with name from assets/branding.

The logo should be large and central.

Headline:

"Praktische 3D-Lösungen. Durchdacht. Robust. Made in Bayern."

Subheadline:

"Elektrozubehör, Skadis-Systeme, Netzwerkhalterungen und individuelle Projekte."

Primary buttons:

* Zum eBay Shop
* Aktuelles ansehen

eBay link:

https://www.ebay.de/usr/slico3d

## Technical Requirements

Create a static website.

Required files:

* index.html
* styles.css
* script.js if needed

Must work on Cloudflare Pages.

Must be responsive.

Must support mobile, tablet and desktop.

Must load fast.

Use semantic HTML.

Use accessible contrast.

Use no external frameworks unless necessary.
